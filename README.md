## âš ï¸ Known Limitation (Important)

The live demo may sometimes show a **timeout error**.

### Reason
- This is currently a **frontend-only application**
- GitHub allows only **60 unauthenticated API requests/hour**
- Evaluating a repository requires multiple API calls

### Production-Ready Solution
- Add a **backend proxy (Node.js / Flask)**
- Store GitHub token securely using environment variables
- Increase rate limit to **5000 requests/hour**

This limitation is acknowledged and addressed at the system design level.
