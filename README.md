GitGrade is an intelligent system that evaluates GitHub repositories and converts them into:

1. A meaningful **SCORE / RATING** (0‚Äì100, Bronze / Silver / Gold)  
2. A concise **WRITTEN SUMMARY** highlighting strengths and weaknesses

The goal is to help **students, recruiters, mentors, and hackathon evaluators** quickly understand the real quality of a GitHub project.

## ‚ö†Ô∏è Known Limitation (Important)

The live demo may sometimes show a **timeout error**.

### Reason
- This is currently a **frontend-only application**
- GitHub allows only **60 unauthenticated API requests/hour**
- Evaluating a repository requires multiple API calls

### Production-Ready Solution
- Add a **backend proxy (Node.js / Flask)**
- Store GitHub token securely using environment variables
- Increase rate limit to **5000 requests/hour**

This limitation is acknowledged and addressed at the system design level.

---

## üîê Security Considerations
- No API keys are exposed in the frontend
- Tokens are intentionally not hard-coded
- Backend-based secure token handling is recommended
